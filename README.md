# JARVIS Voice Assistant

A Python-based voice assistant that uses speech recognition and text-to-speech
to allow users to search Wikipedia using voice commands.

## Features
- Voice-controlled menu system
- Wikipedia voice search
- Text-to-speech responses
- Modular OOP design

## Technologies
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API

## How To Run
pip install -r requirements.txt
python src/jarvis.py
